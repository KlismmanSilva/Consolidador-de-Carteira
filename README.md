# Consolidador de Carteira de Investimentos 📈

Um projeto Full-Stack focado no controle e acompanhamento de ativos financeiros, desenvolvido com FastAPI (Backend), React (Frontend) e SQLite.

## 🚀 Funcionalidades
* Cadastro e listagem de transações financeiras.
* Banco de dados relacional para gerenciar Ativos e Compras.
* Consulta de cotações em tempo real integrado à bolsa de valores via `yfinance`.
* Dashboard visual com cálculo automático de patrimônio, lucro e prejuízo.
